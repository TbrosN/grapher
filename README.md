# Grapher #
A simple graphing calculator.

## Features ##
- Supports operations `+, -, *, /`
- Supports decimals like `0.5` or `.532`
- Supports negative numbers like `-15` or things like `100 + -x`
- Supports implied multiplication as in `5(100-x)(6x+1)`
